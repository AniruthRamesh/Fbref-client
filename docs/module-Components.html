Module: Components

This module consists of reusable UI components designed to facilitate the construction of the application's user interface. It includes components such as Dropdown for selecting from predefined options and RenderStatsRow for displaying player statistics in a tabular format. These components are designed with flexibility and reusability in mind, allowing them to be easily integrated across different parts of the application.

The module plays a crucial role in maintaining a consistent look and feel across the application, providing a centralized location for UI components that can be enhanced and extended as needed.

Future Improvements:

  • Abstract Table Component: Plans are in place to abstract the current table presentation logic into a more generalized Table component. This component will be designed to accept any type of data, rendering it according to a common theme while still allowing for customization as required by different parts of the application. This abstraction aims to reduce redundancy and streamline the process of displaying tabular data, ensuring consistency and efficiency in the development of future features.

Components:

  • Dropdown: Renders a selectable dropdown list, allowing users to choose from a set of predefined options. It's utilized throughout the application for filtering data based on specific criteria such as player names, team types, scopes of statistics, and competition formats.

  • RenderStatsRow: Dynamically generates table rows based on sorted player statistics data. It includes conditional formatting for data presentation and an optional summary row to display aggregate statistics. This component exemplifies the application's data-driven approach to UI design, ensuring information is presented in a clear, concise, and visually appealing manner.

(static) Dropdown(props) → {React.Component}

A Dropdown component that renders a selectable dropdown list. It is utilized throughout the application to offer users a consistent means of selecting from a predefined list of options, such as player names, team types, scopes of statistics, and competition formats.

This component plays a crucial role in the user interface by providing a streamlined way to filter player statistics based on the selected criteria. The dropdown's options are dynamically populated based on the props passed to it, allowing for flexible use across different contexts within the application.

Parameters:
Name Type Description
props Object

The properties passed to the Dropdown component.

Properties
Name Type Description
options Array.<string>

An array of options to be displayed in the dropdown. Each option is a string that represents a selectable item in the dropdown list.
selectedValue string

The currently selected value in the dropdown. This should correspond to one of the strings in the options array.
onChange function

A callback function that is called when the selected option in the dropdown changes. It receives the new selected value as its only argument.

(static) RenderStatsRow(params) → {Array.<React.Component>}

Renders table rows displaying player statistics based on sorted and summarized data. This component is a key part of the application's data presentation layer, transforming structured statistical data into a user-friendly tabular format. Each row represents an individual player's season statistics, with a summary row optionally appended to provide aggregate data.

Parameters:
Name Type Description
params Object

The parameters passed to the RenderStatsRow component.

Properties
Name Type Description
sortedData Array.<Object>

An array of objects, each representing a player's statistics for a single season. This data is pre-sorted based on a selected criterion (e.g., ascending or descending by seasons) in the parent component before being passed to RenderStatsRow.
data Object

An object containing additional data related to the player statistics, including a StatsSummary object that provides aggregate statistics across all listed seasons.
